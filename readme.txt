Instructions to set up and run the application locally
clone the repo 
run npm i command
run node server.js

get all books :  http://localhost:3000 


get book by id  : http://localhost:3000/:id 
                  http://localhost:3000/6540d1ac4af3473edc1f41a7


add book        :  http://localhost:3000  with post method  add data in body

delete book	:  http://localhost:3000/:id   with delete method  


edit book       :    http://localhost:3000/:id     with put method  add data in body




/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

deploy api end points 

get all books :  https://books-apis.onrender.com  


get book by id  :  https://books-apis.onrender.com/:id 
                   https://books-apis.onrender.com/6540d1ac4af3473edc1f41a7


add book        :   https://books-apis.onrender.com  with post method  add data in body

delete book	:   https://books-apis.onrender.com/:id   with delete method  


edit book       :    https://books-apis.onrender.com/:id     with put method  add data in body
